# /assets/images/

This folder contains images used in the portfolio.

## Recommended files to add:

| File                  | Usage                                        | Suggested Size     |
|-----------------------|----------------------------------------------|--------------------|
| `profile.jpg`         | Profile / avatar photo                       | 400×400 px         |
| `og-cover.png`        | Open Graph / social share preview image      | 1200×630 px        |
| `project-truthlens.png` | Screenshot of the TruthLens project        | 800×500 px         |
| `project-airobot.png`  | Photo/screenshot of the AI Robot project   | 800×500 px         |

## Tips
- Use `.webp` format for best performance on modern browsers.
- Keep images under 300 KB where possible — compress with https://squoosh.app
- Use descriptive filenames for SEO (e.g. `aashutosh-gupta-portfolio.jpg` not `img1.jpg`)
