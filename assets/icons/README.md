# /assets/icons/

This folder contains SVG icons and PWA app icons.

## Files included:
- `favicon.svg`   — SVG favicon (works in modern browsers)
- `github.svg`    — GitHub icon (gold stroke)
- `linkedin.svg`  — LinkedIn icon (gold stroke)
- `email.svg`     — Email / mail icon (gold stroke)

## Files to add manually:
| File             | Required For                   | How to generate              |
|------------------|--------------------------------|------------------------------|
| `favicon-32.png` | Older browser favicon fallback | Export favicon.svg at 32×32  |
| `icon-192.png`   | PWA / Android home screen icon | Export favicon.svg at 192×192|
| `icon-512.png`   | PWA splash screen              | Export favicon.svg at 512×512|

### Quick generation tool:
Use https://realfavicongenerator.net to upload your SVG and generate all sizes automatically.
